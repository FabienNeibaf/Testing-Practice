# :robot: Testing Practice

The project main goal is to give a first glance on how javascript tests are handled.  
For that we use the robust and well tested:wink: framework, which is [Jest](https://jestjs.io/).  
This project is part of [microverse](https://www.microverse.org/) curriculum.

# :electric_plug: Set up

To run the project:

- Clone the repository from: _https://github.com/FabienNeibaf/Weather-App_
- Run `npm install`. It will fetch all the dependencies required by the project
- Run `npm run test` and `voil√†!`. Jest will do its magic and logs the details about your tests.
